# ZemTab Logo Main

This folder keeps the main porcelain/coral logo exports and editable source files together.

## PNG exports

- `zemtab-icon-transparent-porcelain-coral.png`
- `zemtab-full-transparent-porcelain-coral.png`
- `zemtab-text-transparent-porcelain-coral.png`

## Editable source

- `zemtab-icon-source.svg`
- `zemtab-logo-source.svg`
- `zemtab-text-source.svg`

To recolor the SVGs, edit the CSS variables near the top of each file:

```css
--coral: #ff4b47;
--porcelain: #f7f1ea;
--dark: #242424;
```

The source files were recreated as clean editable SVGs because no original vector source file was present in the project for the porcelain/coral PNGs.
